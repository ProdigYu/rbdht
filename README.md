## rbdht
A DHT crawler, is not quite based on Kademlia, written by Ruby, the code is very very simple.

## Dependencies
```bash
cd  /path/to/rbdht
gem install bundle
bundle install
```

## Start rbdht
```bash
ruby rbdht.rb
```
